Projects:

Curb Appeal (US County App)

A modern React application that utilizes a flask API to view and compare real estate data with interactive, up-to-date graphs. Leveraged Realtor.com open data library to deserialize U.S. county data and create a Flask API that retrieves the transformed data from a PostgresQL database. Implemented CI/CD pipelines using GitHub Actions to automate testing, building, and deployment.

Weather App

This easy-to-use weather application utilizes the Open Weather API to display the High, Low, Forecast and Humidity of a given location.

SpringBoot User API

Springboot API that allows for registering new users, authenticating, and seeing/editing existing users.

Concert Event Signup

A complete event sign-up page for a fictional event. This project demonstrates the harmonious balance of HTML forms, basic CSS styling, and Git workflow.

News Site

This is a modern, fictional, responsive news magazine. This project showcases proper layout and up to date stories while silmultaneously focusing on easy navigation. The visual categorical separation is created with a combination of CSS Grid for page layout, Flexbox for components, and additional advanced responsive design techniques.
